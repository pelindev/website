# Simple Website for Dev Feature test

## Select project from `./www` folder

```
./www
```
To run the project copy the path as: 
```
/var/www/app
```
to this directory to this file `app.conf` to the `root` line

```
/hosts/app.conf
```
like this:
```
root /var/www/app
```

Run:
```
docker-compose up
```
url:

http://localhost:9000
